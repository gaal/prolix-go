Prolix trims outputs from chatty commands.

See the [online documentation](http://godoc.org/github.com/gaal/prolix-go)
or run `go doc github.com/gaal/prolix-go`.

Prolix is written by Gaal Yahas <gaal@forum2.org>.  
Patches/forks are welcome; please see the code at
[https://github.com/gaal/prolix-go](https://github.com/gaal/prolix-go).
